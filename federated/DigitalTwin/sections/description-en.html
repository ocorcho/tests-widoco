This ontology only provides two additions, for the time being, to the corresponding ontology in FEDeRATED, namely the addition of the concept CMR as a type of Document that is handled in the context of road transport, and the property of relatedDocumentation to allow adding a reference from any business transaction to a document that provides additional information to it.

Examples of usage of many of these classes and properties can be found on the SIMPLE ontology network documentation.
