This ontology extends the Event ontology defined in FEDeRATED with three types of events (represented as classes, as can be seen in the figure).

  • GateInEvent and GateOutEvent, which allow describing the events in which a vehicle enters or departs from a cargo terminal.

  • IssueEvent, which allows representing any type of issue that may happen during the logistics process

Examples of usage of many of these classes and properties can be found on the SIMPLE ontology network documentation.
